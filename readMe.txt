Ordnerstruktur:

WebtechProjekt: In diesm Ordner befinden Sie sich momentan. 
		Er enthält alle .html Dateien, sowie die Ordner "css", "img", "js" und "json" und die readMe-Datei.
css: Beinhaltet die .css Datei.
img: Beinhaltet alle verwendeten Bilder.
js: Beinhaltet alle Java Script Dateien.
json: Beinhaltet eine .json Datei.


Ausführung:

Als Startseite ist die Datei "home.html" angedacht. 
Führen Sie diese je nach gewähltem Standartprogramm zur Öffnung von .html Dateien entweder mit einem Doppelklick aus oder klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie im Dropdownmenü "Öffnen mit Firefox".
